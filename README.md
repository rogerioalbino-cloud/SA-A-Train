# Descrição do Sistema

O **Ferrorama** é um sistema web desenvolvido para o monitoramento inteligente de frotas ferroviárias. Seu objetivo é permitir que operadores e técnicos acompanhem informações sobre os trens, consultem indicadores da operação ferroviária e visualizem relatórios de ocorrências.

A interface foi desenvolvida utilizando **HTML** e **CSS**, seguindo o layout criado no mockup do projeto, oferecendo uma navegação simples, organizada e intuitiva.

---

# Páginas Implementadas

O sistema é composto pelas seguintes páginas:

- ✅ Login
- ✅ Cadastro de Usuário
- ✅ Dashboard
- ✅ Lista de Trens
- ✅ Relatórios

# Como Executar o Projeto

1. Faça o download ou clone este repositório:

```bash
https://github.com/rogerioalbino-cloud/SA-A-TRAIN

```

2. Abra a pasta do projeto no **Visual Studio Code**.

3. Navegue até o arquivo **index.html**.

4. Abra o arquivo utilizando a extensão **Live Server** ou diretamente em um navegador de internet.

5. O sistema será iniciado pela tela de Login.
